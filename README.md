# SILO – Shorttrack data tool

Static web app (no backend) that loads an Excel file locally and provides modules:
- Dashboard
- Filters & Parameters
- Head-to-Head
- Champions (WK/OS)
- Overview
- Settings (mapping + data controls)

## Run locally
Use a local server (recommended):
- VS Code → Live Server
- or `python -m http.server` and open http://localhost:8000
